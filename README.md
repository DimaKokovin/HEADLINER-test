# 🛒 Carts Management SPA

Single Page Application для работы с корзинами пользователей на основе публичного API DummyJSON.

Тестовое задание выполнено с использованием React, TypeScript и React Query.

## 📌 Функциональность

### 📋 Страница списка корзин
- Отображение списка корзин
- Пагинация (`limit / skip`)
- Loading / Error состояния
- Сохранение текущей страницы при возврате
- Плавное переключение страниц (placeholderData)

### 📦 Детальная страница корзины
- ID корзины
- userId
- Список товаров
- Цена, количество, итог по товару
- Общая сумма корзины
- Loading / Error состояния

### ✏ Редактирование корзины
- Изменение количества товара
- Удаление товара
- Optimistic update
- Блокировка UI во время обновления
- Автоматическая инвалидация кэша
- Обработка ошибок с rollback

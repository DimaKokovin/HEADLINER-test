# Frontend Test Task – React + React Query

## 📌 Описание проекта
SPA-приложение для работы с корзинами пользователей на основе публичного API [DummyJSON](https://dummyjson.com/docs/carts).

Приложение позволяет:

- Просматривать список корзин с пагинацией
- Переходить в детали конкретной корзины
- Изменять количество товаров или удалять их
- Сохранять состояние пагинации при навигации

> Все изменения сохраняются локально в UI через optimistic update. API DummyJSON является mock-сервисом и не сохраняет изменения на сервере.

**Демо:** [Рабочий деплой на Vercel](https://headliner-test.vercel.app/)

---

## 🗂 Архитектура проекта

- **entities/cart/api.ts** — функции работы с API (GET /carts, GET /carts/:id, PUT /carts/:id)
- **store/usePaginationStore.ts** — глобальный стейт для текущей страницы и limit
- **pages/** — UI страниц
- **App.tsx / main.tsx** — BrowserRouter + QueryClientProvider

---

## 🚀 Функциональные требования

1. **Список корзин**
    - ID корзины, userId, количество товаров, общая сумма
    - Пагинация (limit/skip)
    - Loading / Error states
    - Состояние сохраняется при возврате со страницы деталей

2. **Детальная страница корзины**
    - ID корзины, userId
    - Таблица с товарами: название, цена, количество, total
    - Общая сумма корзины
    - Возможность изменить количество или удалить товар
    - Optimistic update для мгновенного обновления UI

3. **Редактирование корзины**
    - Изменение количества товаров
    - Удаление товара
    - Автоматическое обновление списка корзин и детальной страницы

4. **Глобальный стейт**
    - Сохраняет текущую страницу пагинации и limit
    - (Опционально) фильтрация по userId
